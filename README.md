### Technologies used
* HTML5 (only one file)
* [Bootstrap 5.1.3](https://getbootstrap.com)
* JavaScript (SPA)

### Challenge
Code a funcional quiz:
* With 10 questions
* Each question with 4 possible answers
* As a Single Page Application
* Fetching questions from an API
* Storing results in localStorage
